Setup the Boilerplate
================================

rename local\_settings.example.py to local\_settings.py and alter this file to your needs and setup



```bash
./manage.py syncdb
./manage.py migrate
./manage.py runserver
```

* site: 127.0.0.1:8000/
* admin: 127.0.0.1:8000/admin
* documentation: 127.0.0.1:8000/doc/index.html
